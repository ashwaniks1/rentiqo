# Rentiqo Backend

Stage 4 backend scaffold for the Rentiqo platform.

## Commands

- `npm run dev -w @rentiqo/backend`
- `npm run typecheck -w @rentiqo/backend`
- `npm run build -w @rentiqo/backend`

## Implemented scaffold routes

- `GET /v1/health`
- `POST /v1/search/listings`
- `GET /v1/listings/:listingId`
