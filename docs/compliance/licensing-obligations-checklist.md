# Licensing Obligations Checklist (MLS/IDX and Data Partners)

## 1) Pre-onboarding checks

- [ ] License agreement signed for launch market.
- [ ] Data usage restrictions documented and accepted.
- [ ] Attribution language and display requirements documented.
- [ ] Prohibited use cases mapped to product guardrails.

## 2) Engineering compliance checks

- [ ] Source identifiers preserved in canonical model.
- [ ] Display attribution implemented in listing surfaces where required.
- [ ] Data refresh cadence meets contractual obligations.
- [ ] Retention/deletion behavior aligned with source terms.

## 3) Operations checks

- [ ] Source change notifications monitored.
- [ ] Compliance owner assigned per source relationship.
- [ ] Periodic internal review scheduled.
- [ ] Incident escalation path defined for source-policy violations.

## 4) Audit evidence

- [ ] Mapping from license clauses -> technical controls.
- [ ] Screenshots/evidence for attribution implementation.
- [ ] Logs showing refresh and retention policy enforcement.
