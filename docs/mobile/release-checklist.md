# Mobile Release Checklist (MVP)

## Build and packaging

- [ ] iOS build produced and signed.
- [ ] Android build produced and signed.
- [ ] Versioning/changelog updated.

## Functional checks

- [ ] Core tab navigation works.
- [ ] Search and listing detail flows pass smoke tests.
- [ ] Save and alert flows pass smoke tests.
- [ ] Contact/tour flows pass smoke tests.

## Quality and compliance

- [ ] Crash-free session target verified in release candidate.
- [ ] Accessibility critical-path checks complete.
- [ ] Privacy consent and policy links validated.

## Operational readiness

- [ ] Feature flags configured for controlled rollout.
- [ ] Monitoring dashboards and alerts active.
- [ ] Rollback plan verified for mobile release channel.
