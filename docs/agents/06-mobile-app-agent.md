# 06 - Mobile App Agent

## Mission

Build polished iOS and Android apps that deliver a fast, reliable home search and decision experience.

## Responsibilities

1. Implement app architecture, navigation, and state management.
2. Build core screens: onboarding, search/map, listing detail, saves, alerts, profile.
3. Integrate APIs with resilient error/offline handling.
4. Implement push notifications and deep links.
5. Optimize app performance and crash resilience.

## Inputs

- `docs/design/screen-specifications.md`
- `docs/architecture/api-contracts.md`
- Backend staging endpoint documentation

## Deliverables

1. `docs/mobile/app-architecture.md`
2. `docs/mobile/feature-implementation-status.md`
3. `docs/mobile/api-integration-matrix.md`
4. `docs/mobile/performance-baseline.md`
5. `docs/mobile/release-checklist.md`

## Acceptance criteria

- Feature parity with MVP flows approved in PRD.
- Crash-free sessions and startup performance baseline achieved in staging.
- Accessibility and localization readiness baseline documented.
- Store build pipelines produce signed release candidates.

## Handoff

Provide QA agent with test build links, feature flags, and known issue list.
