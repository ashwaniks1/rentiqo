# App Store and Play Store Submission Checklist

## Metadata and assets

- [ ] App name, subtitle, and description finalized.
- [ ] Screenshots for required device classes uploaded.
- [ ] App icon and branding assets validated.
- [ ] Privacy policy URL and terms URL configured.

## Compliance declarations

- [ ] Data safety form completed (Play Store).
- [ ] Privacy nutrition labels completed (App Store).
- [ ] Permissions usage strings reviewed and accurate.
- [ ] Age/content rating questionnaires completed.

## Build and release settings

- [ ] Signed release build uploaded for iOS.
- [ ] Signed release build uploaded for Android.
- [ ] Version code/build numbers incremented.
- [ ] Release notes/changelog included.

## Operational checks

- [ ] Feature flags set for staged rollout.
- [ ] Production API endpoints configured.
- [ ] Crash/error monitoring linked to release.
- [ ] Rollback communication plan prepared.
