# Lifecycle Engagement Playbook

## 1) Lifecycle stages

- New user (day 0-3)
- Active evaluator (day 4-30)
- Dormant user (no meaningful action in window)
- Re-engaged user

## 2) Triggered engagement programs

### New user activation
- Welcome sequence and search setup prompt.
- Nudge to save first home/search.

### Active evaluator support
- New/updated listing alerts tuned to preferences.
- Affordability and comparison nudges.

### Dormant reactivation
- Digest of fresh relevant listings.
- Re-entry flow with updated preference capture.

## 3) Messaging channels

- Push notifications (primary)
- Email digests (secondary)
- In-app nudges and banners

## 4) Guardrails

- Frequency caps by channel and user segment.
- Respect notification and consent preferences.
- Suppress redundant alerts using dedupe keys.

## 5) KPIs

- Alert open rate
- Re-engagement rate
- Conversion from reactivation to save/contact actions
