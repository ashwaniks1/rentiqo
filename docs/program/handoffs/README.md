# Handoffs Directory

This directory stores task-to-task and agent-to-agent handoff records.

## Naming format

`<task-id>-<from-agent>-to-<to-agent>-<YYYYMMDDHHMM>.md`

Example:

`A1-backend-agent-to-mobile-agent-202605220930.md`

## Required content

Use `docs/program/handoff-template.md` for every handoff file.
