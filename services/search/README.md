# Rentiqo Search Service

Stage 4 scaffold for ranking, saved-search matching, and relevance evaluation.

## Commands

- `npm run dev -w @rentiqo/search-service`
- `npm run typecheck -w @rentiqo/search-service`

## Modules

- `ranking/ranker.ts` - baseline ranking scorer
- `alerts/matcher.ts` - saved search alert matcher scaffold
- `evaluation/baseline.ts` - relevance metrics helper
